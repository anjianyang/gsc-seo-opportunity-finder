# Roadmap

## MVP10 - AI Action Plan

Goal: turn rule-based opportunities into richer SEO action plans with optional AI assistance.

Planned scope:

- Generate stronger page-specific recommendations
- Suggest improved title tag variations
- Suggest meta description variations
- Suggest FAQ ideas
- Keep AI optional and clearly separated from local rule-based analysis

## MVP11 - SEO Report Generator

Goal: improve report quality and make exports more client-ready.

Planned scope:

- Add branded report cover page
- Add report sections by page
- Add downloadable HTML report
- Add improved PDF formatting
- Add summary recommendations for non-technical users

## MVP12 - SaaS Version

Goal: evolve the local MVP into a hosted SaaS product.

Planned scope:

- Add user accounts
- Add project history
- Add saved reports
- Add payment flow
- Add subscription tiers
- Add optional Google Search Console API integration
- Add multi-site management
